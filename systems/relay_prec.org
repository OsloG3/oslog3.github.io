* Relays and Parity cuebidding
*** 5 Card scheme (5-CS)
This is used when we have shown 5 card or longer suit. and partner relays with 1NT.\\
Over a 1!s opening all the responses will be one step higher. In all other cases they are as below\\
With 2 suiter, 5332 and 5440 shapes we only start by showing the exact shape, while with single suiters we show min/max before showing the exact shape. This is done to increase the symmetry of the relay structure.\\
The sequences only include the bids of the player showing steps assuming his partner keeps relaying\\
In general we use the phone number rule, if writing our shape in order SHDC higher numbers will bid higher. This doesn't apply to 55s/7411/5440 shapes
- 2!c :: 4+Lower or 5332 or 6+X min (step-1)
  - 2!h :: 5332 or 6+X min (even number of side suits)
  - 2!s+ :: 4+Lower
- 2!d :: 4+Middle (step-2)
- 2!h :: 5X(440) or max 6+X (even number of side suits) (step-3)
- 2!s+ :: 4+Higher (step-4)

*** No 5+suit relay (No5R)
It might look a bit messy at first glance. But we actually only need to remeber tha\\
1) bid 2NT iff 4441 \\
2) bid 3!d iff 4333 \\

Than the phone book rule forces how the entire structure has to be.

- 2!s :: 2-3!s, not 3=4=3=3
  - 3!c :: 2!s
    - 3!h :: 2=3=4=4
    - 3!s :: 2=4=3=4
    - 3NT :: 2=4=4=3
  - 3!d :: 4m333
    - 3!h :: shape ask
      - 3!s :: 3=3=3=4
      - 3N :: 3=3=4=3
    - 3!s :: strength ask, I don’t care about shape
  - 3!h :: 3=2=4=4
  - 3!s :: 3=4=2=4
  - 3N :: 3=4=4=2
- 2N :: any 4441 → S/H/D/C short
- 3!c :: 4!s first 3 steps
  - 3!h :: 4=2=3=4
  - 3!s :: 4=2=4=3
  - 3NT :: 4=3=2=4
- 3!d ::  4M333
  - 3!h :: shape ask
    - 3!s :: 3=4=3=3
    - 3N :: 4=3=3=3
  - 3!s :: strength ask, I don’t care about shape
- 3!h :: 4=3=4=2 (4!s, 4th shape)
- 3!s :: 4=4=2=3 (4!s 5th shape)
- 3N :: 4=4=3=2 (4!s 6th shape)

 Shapes ending at 3NT will bid 4!c+ if with the 15+ option. Other shapes will bid as normal and later reopen
